# Windows 11 AutoUnattend

Archivo `autounattend.xml` para crear una instalación desatendida de Windows 11 con configuración personalizada.

## Qué hace
- Automatiza parte de la instalación de Windows.
- Permite ajustar edición, idioma, usuario local y opciones iniciales.
- Ayuda a crear un USB/ISO de instalación más cómodo para pruebas y despliegues.

## Uso
1. Descarga `autounattend.xml`.
2. Copia el archivo en la raíz del USB de instalación de Windows 11.
3. Arranca el equipo desde el USB.
4. Sigue el asistente automatizado.

## Aviso
Haz siempre copia de seguridad antes de reinstalar Windows. Prueba primero en una máquina virtual.
